### RabbitMQ学习笔记：Connections、Channels、Queues之state状态

##### 连接、信道、队列状态如下：

|    Name     | running(运行中) | flow（流控） | idle（空闲） |
| :---------: | :-------------: | :----------: | :----------: |
| Connections |      true       |     true     |     true     |
|  Channels   |      true       |     true     |     true     |
|    Queue    |      true       |     true     |     true     |
